# Book List 📖

This app allows you to save details of your book list. [View Live App](https://shivams29.github.io/Book-List).

### Features

-   Beautiful UI 🌈
-   Browser Persistance 🌐
-   Easy to use ✅
-   Styled with Bootstrap 💅
-   Powered by Vanilla JavaScript ⚡
-   Hosted on Github :octocat:
